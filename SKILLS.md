# Pangkalan Kemahiran Projek (SKILLS.md)

Memandangkan projek ini diuruskan oleh *Agent AI* merentasi pelayan autonomi, di bawah adalah set *Skill* (kemahiran) rujukan rasmi yang mana setiap AI perlu ada (atau faham) semasa memanipulasi *codebase* ini:

## Kemahiran Teras (Core AI Skills Required)

* **Vanilla JS Mastery**: Pemahaman mendalam tentang DOM, manipulasi `document.getElementById`, pengurusan tatasusunan (Array), objek, dan `localStorage()`.
* **CSS Architecture**: Pakar menggayakan fail CSS tanpa Tailwind atau Bootstrap. Mahir menggunakan Custom Variable (`var(--)`). Dilarang sepenuhnya mengubah CSS struktur gred projek klien ke teknologi lain.
* **Prompt Engineering & OODA Loop**: (Rujuk kemahiran `ai-orchestration` dan `leo-orchestrator`). Penting sewaktu membaca logik ralat dan membaikinya dengan berhati-hati.

## Rujukan Kemahiran Global (Cross-Reference)

Sekiranya aplikasi *Dashboard Live* ini akan dikembangkan ke infrastruktur pelayan, Ejen sangat disyorkan membaca senarai `SKILL.md` (di ruangan `antigravity\skills`) seperti:

1. **`nodejs-backend`** (Jika projek mahu dikalibrasikan ke pelayan Node/Express masa hadapan).
2. **`database-admin`** (Sekiranya pengguna ingin menggunakan PostgreSQL bagi menggantikan localStorage).
3. **`github-mastery`** (Pengendalian urusan kod selari, PR, dan rilis).
4. **`testing-mastery`** (Membina unit testing untuk memastikan pengiraan pendermaan tidak pernah lari 1 Pax pun).
5. **`ui-ux-promax`** (Garis panduan rasmi sekiranya mahu memastikan sistem glow `overtime-mode` tampak lebih "premium" pada masa kelak).
