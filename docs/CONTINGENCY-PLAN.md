## Pelan Jika Kutipan Perlahan

## SCENARIO 1: 30 Minit Pertama < 20% Target

**Tanda**: Kurang 400 pax dalam 30 minit pertama.

### Tindakan Segera

1. **Sprint 15 Minit** — Set target kecil: "+50 pax dalam 15 minit!"
2. **Fokus Pakej 10 Pax** — Push RM50, bukan pakej besar
3. **Social Proof** — Bacakan semua nama donor yang ada, biar nampak ramai
4. **Share Push** — "Tolong share live ni! Kita perlukan lebih ramai viewer"
5. **Emosi Trigger** — Cerita impak: "1 orang tak dapat makan esok kalau kita tak cukup"

---

## SCENARIO 2: 60 Minit < 40% Target

**Tanda**: Kurang 800 pax dalam 1 jam.

### Tindakan

1. **Extend Live** — Maklum viewer: "Kita sambung sesi kedua selepas rehat 10 minit"
2. **Re-Hook** — Treat seperti live baru, ulang opening hook
3. **Lower Barrier** — Push 1-5 pax: "RM5 pun cukup, jangan rasa kecil"
4. **Urgent CTA** — "Kita baru capai [X]%, perlukan bantuan semua"
5. **Contact Big Donors** — DM donor besar dari sesi sebelum, minta matching

## SCENARIO 3: 90 Minit < 60% Target

**Tanda**: Kurang 1200 pax selepas 1.5 jam.

### Tindakan

1. **Realistic Adjust** — "Alhamdulillah kita dah capai [X] pax. Insya-Allah cukup untuk agihan esok"

5. **Platform Lain** — Cross-post ke Instagram Live, Facebook

## SCENARIO 5: Isu Teknikal

| Masalah | Penyelesaian |
|---------|-------------|
| Live disconnect | Masuk balik segera + pin komen: "Kita dah reconnect!" |
| QR tak boleh scan | Sediakan no. akaun text backup |
| Dashboard error | Refresh browser / guna board fizikal backup |
| Audio problem | Switch ke phone speaker / external mic |
| Bank offline | Guna e-wallet backup (TnG/GrabPay) |

### Backup Plan

- **Sentiasa ada board fizikal** (marker + whiteboard) sebagai backup digital
- **2 device**: 1 untuk live, 1 untuk monitor dashboard
- **QR printed** di kertas A4 sebagai backup

## SCENARIO 6: Viewer Count Rendah

### Jika < 50 Viewers

1. Post story/reel: "LIVE SEKARANG! Jom join!"
2. Minta viewer share ke 3 kawan
3. Gunakan TikTok coin/gift untuk boost visibility
4. Tag lokasi trending
5. Guna hashtag: #BuburLambuk #Ramadan2026 #Sedekah #Infaq

### Jika 50-200 Viewers

- Momentum OK, teruskan push CTA biasa
- Fokus conversion rate, bukan viewer count

### Jika > 200 Viewers

- Peluang besar! Push pakej besar (20/50/100 pax)
- Sprint mode setiap 15 minit

## SCENARIO 7: Target Exceeded (>2000 pax)

**Tanda**: Target 2000 pax tercapai sebelum live habis.

### Tindakan Segera

1. **Celebrate!** — "ALHAMDULILLAH! TARGET TERCAPAI!" + confetti
2. **Announce pilihan** kepada viewer:

### Option A: Scale ke 2500 pax

| Criteria | Tindakan |
|----------|----------|
| Jika > 2200 pax tercapai | Auto-proceed ke 2500 target |
| Jika 2000-2200 pax | Tanya audience: "Nak terus ke 2500?" |
| Set target 2500 | Update dashboard target ke 2500 |

**Logistik tambahan:**

- Beli bahan tambahan esok pagi (rujuk SUKATAN-BAHAN 2000 pax + buffer)
- Tambah 500 plastik bag + bungkusan
- Mungkin perlukan 1 periuk tambahan

### Option B: Carry Forward

| Baki Dana | Tindakan |
|-----------|----------|
| < RM1,000 | Carry forward ke esok sebagai head start |
| RM1,000 - RM2,500 | Esok mulakan dengan 200-500 pax sudah cukup |
| > RM2,500 | Boleh cover 2 hari penuh! |

**Announcement:**

- "Baki RM[X] akan digunakan 100% untuk bubur lambuk hari berikutnya"
- Post laporan ketelusan: berapa masuk, berapa guna, berapa baki

---

## SCENARIO 8: Carry Forward SOP

### Proses

1. **Catat** jumlah akhir dana terkumpul
2. **Tolak** kos bahan hari ini (rujuk TEMPLATE-KEWANGAN)
3. **Baki** = Dana carry forward
4. **Announce** baki di akhir live + post caption
5. **Mulakan** live esok dengan: "Kita dah ada head start [X] pax!"

### Template Announcement

```text
📊 LAPORAN HARI 1:
✅ Terkumpul: RM[TOTAL]
💰 Digunakan: RM[GUNA] (untuk [X] pax)
📦 Baki carry forward: RM[BAKI] (= [Y] pax)
🎯 Esok kita mulakan dengan [Y] pax sudah ada!
```
