## WEEKLY FRAMING

### Minggu 1 (6-12 Mac)

**Tema: "Membangunkan Momentum"**

- Hari 1-2: Target kecil (500-1000 pax/hari)
- Hari 3-5: Naikkan target (1000-1500 pax/hari)
- Hari 6-7: Showcase impak minggu pertama

### Minggu 2 (13-19 Mac)

**Tema: "Rapatkan Saf"**

- Fokus repeat donors
- Introduce "team penyumbang" recognition
- Collab dengan akaun lain untuk reach baru
- Target: 1500-2000 pax/hari

### Minggu 3 (20-28 Mac)

**Tema: "Sprint Akhir Ramadan"**

- Intensity tinggi
- "Setiap hari menghitung!"
- Target: 2000+ pax/hari
- Grand finale malam terakhir

## STRATEGI RETENTION (Donor Repeat)

### Donor Ladder

```
Hari 1: Donor infaq 1 pax (RM5)
    ↓ Terima kasih + sebut nama
Hari 3: Donor infaq 5 pax (RM25)
    ↓ "Terima kasih repeat donor!"
Hari 7: Donor infaq 10 pax (RM50)
    ↓ "Donor minggu ni!"
Hari 14: Donor infaq 20 pax (RM100)
    ↓ "Sahabat kempen!"
Hari 21: Donor infaq 50 pax (RM250)
    ↓ "Champion Ramadan!"
```

### Taktik Retention

1. **Nama Recognition** — Sebut nama setiap kali live
2. **Progress Update** — DM/reply donor dengan update agihan
3. **Milestone Badge** — "5x Donor", "10x Donor" etc
4. **Exclusive Update** — Hantar gambar masak/agihan via DM ke repeat donor
5. **Final Day Special** — Sebut semua nama repeat donor malam terakhir
