# 👥 SOP TEAM — AGENSI STYLE

## Misi 2000 Pax Bubur Lambuk

> **Minimum 4 orang, ideal 5 orang.** Setiap role ada checklist sebelum, semasa, dan selepas live.

---

## ROLE 1: 🎤 HOST (Presenter)

### Tanggungjawab

- Deliver naratif dan skrip live
- Ulang CTA setiap 3-5 minit
- Bacakan nama penyumbang yang telah verified
- Update progress secara vokal
- Handle Q&A ringkas

### Checklist Sebelum Live

- [ ] Baca skrip sekali penuh
- [ ] Hafal opening 20 saat
- [ ] Sediakan air minum
- [ ] Pastikan pencahayaan OK
- [ ] Test audio/mic
- [ ] Pakai baju kempen / baju raya (nampak profesional)

### Semasa Live

- [ ] Ikut skrip minit-per-minit, mula dengan cerita + tunjuk bahan
- [ ] Pantau isyarat dari Mod Chat (nama, pax)
- [ ] **WAJIB:** Sebut nama SETIAP penyumbang bila verified ("Terima kasih X!")
- [ ] Ulang CTA setiap 3-5 minit
- [ ] Jangan biar dead air lebih 10 saat (teruskan sembang/potong bahan)
- [ ] Tunjuk emosi dan keikhlasan (authentic, jangan jadi robot)

### Selepas Live

- [ ] Recap ringkasan kutipan
- [ ] Ucapan terima kasih
- [ ] Maklumkan jadual esok

---

## ROLE 2: 💬 MOD CHAT (Moderator Komen)

### Tanggungjawab

- Pin komen utama
- Kumpul komen format "DONE + pax"
- Reply viewer dengan arahan bayar
- Tapis soalan penting untuk Host
- Forward nama + pax ke Finance

### Checklist Sebelum Live

- [ ] Salin pin komen (ready paste)
- [ ] Buka sheet tracking
- [ ] Login sebagai moderator TikTok
- [ ] Sediakan template reply

### Template Reply

```
Terima kasih! Sila transfer RM[X] ke [AKAUN/QR] dan hantar screenshot DM. ✅
```

```
VERIFIED ✅ Terima kasih [NAMA]! [X] pax telah direkodkan.
```

```
Niat diterima insya-Allah. 1 pax = RM5. Tulis DONE + jumlah pax.
```

### Semasa Live

- [ ] Pin komen setiap 10 minit (refresh info & akaun bank supaya sentiasa atas)
- [ ] Kumpul semua "DONE" komen
- [ ] Reply "Terima kasih, sila transfer & hantar rujukan"
- [ ] Forward nama ke Finance untuk verification
- [ ] **CRITICAL:** Tulis nama penyumbang yang dah verified di papan putih/kertas besar dan tunjuk pada Host untuk dia bacakan
- [ ] Whistle/isyarat ke Host bila ada donor besar masuk

### Selepas Live

- [ ] Pastikan semua "DONE" telah di-forward ke Finance
- [ ] Screenshot final komen untuk rekod

---

## ROLE 3: 💰 FINANCE VERIFIER

### Tanggungjawab

- Semak transaksi masuk dalam akaun
- Verify jumlah tepat
- Update status "VERIFIED" dalam sheet
- Maklumkan Mod Chat untuk reply
- Maklumkan Scoreboard untuk update

### Checklist Sebelum Live

- [ ] Login banking app / e-wallet
- [ ] Buka sheet tracking
- [ ] Sediakan QR payment untuk display
- [ ] Test terima RM1 sendiri (pastikan QR berfungsi)

### Flow Verification

```
1. Mod Chat forward: "[NAMA] - DONE 10"
2. Finance semak akaun: RM50 masuk? ✅
3. Finance update sheet: Status = VERIFIED
4. Finance maklum Mod Chat: "VERIFIED ✅"
5. Mod Chat reply viewer: "VERIFIED ✅"
6. Finance maklum Host: "Sebut [NAMA] 10 pax"
```

### Semasa Live

- [ ] Refresh banking setiap 2-3 minit
- [ ] Match transaksi dengan komen
- [ ] Tandakan VERIFIED dalam sheet
- [ ] Maklumkan jumlah terkini setiap 5 minit

### Selepas Live

- [ ] Tally final semua transaksi
- [ ] Pastikan total sheet = total akaun
- [ ] Screenshot bukti transaksi akhir
- [ ] Simpan untuk laporan esok

---

## ROLE 4: 📊 SCOREBOARD OPERATOR

### Tanggungjawab

- Update dashboard/board real-time
- Pastikan nombor tepat on-screen
- Update setiap 5-10 minit atau setiap donor baru

### Checklist Sebelum Live

- [ ] Buka dashboard (index.html) dalam browser
- [ ] Set target: 2000 pax
- [ ] Set semasa: 500 pax
- [ ] Pastikan nampak jelas dalam frame live

### Semasa Live

- [ ] Terima update dari Finance
- [ ] Tambah donor dalam admin panel
- [ ] Pastikan progress ring & stats sentiasa updated
- [ ] Alert Host bila capai milestone (600, 700, 800... 2000)

### Selepas Live

- [ ] Export CSV sebagai rekod rasmi
- [ ] Screenshot final dashboard

---

## ROLE 5: 📸 OPS VISUAL (Runner/Camera)

### Tanggungjawab

- Tunjuk bahan fizikal on-camera
- Rakam video bukti prep
- Sediakan visual trust (resit, bahan, lokasi)
- Dokumentasi untuk post esok

### Checklist Sebelum Live

- [ ] Susun bahan atas meja (cantik & teratur)
- [ ] Lighting cukup untuk tunjuk bahan
- [ ] Sediakan resit pembelian bahan
- [ ] Rakam video 30s bahan sebagai backup
- [ ] Print/tulis board fizikal (backup jika digital fail)

### Semasa Live

- [ ] Tunjuk bahan bila Host cue (M10-15, M80-85)
- [ ] Tunjuk resit bila Host cue (M45-50)
- [ ] Rakam behind-the-scenes content
- [ ] Sediakan bahan untuk sesi close-up

### Selepas Live

- [ ] Kemas bahan, simpan dalam peti sejuk (bahan segar)
- [ ] Compile semua video/foto untuk post esok
- [ ] Hantar ke Host/social media manager

---

## 📋 COMMUNICATION PROTOCOL

| Signal | Maksud |
|--------|--------|
| 🖐️ (5 jari) | "5 minit lagi milestone" |
| ✊ (penumbuk) | "Donor besar masuk!" |
| 👆 (tunjuk atas) | "Update nombor on screen" |
| ✌️ (peace sign) | "2 minit lagi tutup section" |
| 🤫 (shhh) | "Biar Host cakap, jangan potong" |
